# WaelderCPU

This project is a diploma thesis done at the HTBLUVA Rankweil. It's purpose lies within explaining how a CPU works and making one ourselfes in order to abstract the complex theroy behind processing units.  

This GitHub is currently still under Developement but will be updated in the future.  

The end of the diploma thesis is not the end of the project, it is aimed to be expanded in the future.
